# Project README

## Setup

This project has separate **backend** and **frontend** directories. Follow these steps to set up the project:

1. **Backend**

```bash
cd backend
npm i
```

- Create a `.env` file in the backend directory using the example in `.env.exmaple`.

2. **Frontend**

```bash
cd frontend
npm i
```

- Create a `.env` file in the frontend directory using the example in `.env.exmaple`.

## Running the Project

1. **Backend**

```bash
cd backend
npm run start:dev
```

2. **Frontend**

```bash
cd frontend
npm run dev
```
